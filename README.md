# ⚓ Sakuga Piece — Архив анимации One Piece

Сайт-архив сакуга-моментов (моментов с крутой анимацией) из One Piece.  
Поиск по аниматорам, аркам, эпизодам. Загрузка клипов в высоком качестве.

---

## 📁 Структура проекта

```
sakuga-piece/
├── server.js          # Node.js сервер (Express)
├── package.json       # Зависимости
├── clips.json         # База данных клипов (создаётся автоматически)
├── uploads/           # Загруженные видеофайлы
├── public/            # Фронтенд
│   ├── index.html     # Главная страница
│   ├── style.css      # Стили
│   └── app.js         # Клиентский JavaScript
└── README.md          # Этот файл
```

---

## 🚀 Запуск (локально)

### 1. Установите Node.js
Скачайте и установите с https://nodejs.org (версия 18+)

### 2. Установите зависимости
```bash
cd sakuga-piece
npm install
```

### 3. Запустите сервер
```bash
npm start
```

Сайт будет доступен по адресу: **http://localhost:3000**

## 🌐 Как сделать доступным для всех в интернете

Есть несколько вариантов:

### Вариант 1: Railway (бесплатно для начала)
1. Зарегистрируйтесь на https://railway.app
2. Создайте новый проект → Deploy from GitHub
3. Загрузите папку проекта в GitHub репозиторий
4. Railway автоматически определит Node.js и запустит сервер
5. Вы получите публичный URL вида `sakuga-piece.up.railway.app`

### Вариант 2: Render (бесплатно)
1. Зарегистрируйтесь на https://render.com
2. New → Web Service → подключите GitHub репозиторий
3. Build Command: `npm install`
4. Start Command: `npm start`

### Вариант 3: VPS (полный контроль)
1. Арендуйте VPS (Hetzner, DigitalOcean, и т.д.)
2. Установите Node.js на сервере
3. Скопируйте проект на сервер
4. Запустите через pm2:
```bash
npm install -g pm2
pm2 start server.js --name sakuga-piece
```
5. Настройте Nginx как reverse proxy и SSL-сертификат

## ⚠️ Важно для продакшена

- **Хранение видео**: Для большого количества клипов стоит подключить облачное хранилище (AWS S3, Cloudflare R2)
- **Домен**: Купите домен и настройте DNS на ваш сервер
- **Бекапы**: Регулярно делайте бекапы `clips.json` и папки `uploads/`
- **Защита**: Для реального продакшена стоит добавить авторизацию для загрузки клипов

## 🎮 Использование

- **Поиск**: Введите имя аниматора, номер эпизода, название арки или тег
- **Фильтры**: Нажмите на чипы вверху для фильтрации по категории или арке
- **Загрузка**: Нажмите «+ Загрузить клип» и заполните форму
- **Просмотр**: Нажмите на карточку клипа для воспроизведения
- **Горячие клавиши**: `/` — фокус на поиске, `Esc` — закрыть окна
